static int32_t cia402_master_get_target ( void  )
static

APPLICATION POINT – returns the target position for the current SYNC cycle.

Called on every FSM iteration while in CIA402_M_RUNNING state. Modify this function to implement custom motion profiles.

Default behaviour: toggles between 0 and 100 pulses every 5000 SYNC frames.

Returns
Target position [pulses] to be sent to the drive.

SDO download confirmation callback.

Registered as the completion handler for co_csdo_dn_val_req() in the CIA402_M_SDO_INIT state. On success sets sdo_done so the FSM can advance to CIA402_M_SHUTDOWN. On failure logs the abort code and returns the FSM to CIA402_M_IDLE.

Parameters
[in]sdoClient SDO object (unused).
[in]idxOD index of the downloaded object (unused).
[in]subidxOD sub-index of the downloaded object (unused).
[in]acSDO abort code; 0 on success.
[in]dataUser data pointer (unused).
