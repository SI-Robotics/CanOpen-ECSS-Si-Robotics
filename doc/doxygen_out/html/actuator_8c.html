Detailed Description

Implementation of actuator cache layer for CiA-402 over CANopen (RPDO1/TPDO1).

This module keeps a local cache for both MASTER and SLAVE roles:

  • MASTER: set Controlword (0x6040) + Target position (0x607A), transmit RPDO1, parse TPDO1 (Statusword 0x6041 + Position actual 0x6064).
  • SLAVE : receive CW/Target via OD pull (RPDO1 mapped objects), transmit SW/ActualPos via OD push + TPDO event.

The actual node_id/bus_id configuration and PDO mapping are handled by cia402_wrapper.

const uint8_t *  data,

Process a received CAN frame on the master side (TPDO1).

The frame is forwarded to the wrapper parser. If it matches TPDO1 for the configured node (typically COB-ID = 0x180 + node, mapping 0x6041:16 + 0x6064:32), the wrapper updates its values and this module pulls them into the master cache.

Parameters
cob_idCAN identifier (COB-ID).
dataPayload pointer.
lenPayload length (expected 6 bytes for 16+32 mapping).
